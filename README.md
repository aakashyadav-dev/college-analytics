# 🎓 College Analytics

A full-stack web application to manage and analyze college data efficiently.

---

## 🚀 Features

- 📊 Manage college data (add, update, delete)
- 🔗 REST API built with Node.js & Express
- 🌐 Simple frontend using HTML, CSS, JavaScript
- 📁 Organized backend and frontend structure

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
